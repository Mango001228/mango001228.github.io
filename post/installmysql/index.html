安装与配置MySQL

Aug 5, 2022
1 min read

问题汇总：

  1. net start mysql 发生错误

安装MySQL：

  1. 官方下载所需版本，双击启动安装程序
  2. 点击developer default ，再next，再点击back（帮助选择所需开发工具），再选择custom（经典自定义安装），更改install directory和data directory，选择自己所需位置，不建议默认安装至c盘，可能会导致权限问题
  3. 一键next+execute，设置数据库密码，最后check，确保不出错
  4. 在mysql/bin目录下cmd，mysql -u root -p 登录mysql成功后，再验证版本号mysql –version，能正确显示即安装成功

配置环境：

  1. 系统变量Path中，新增MySQL bin地址即可 ，例：D:\MySQL\MySQL Server 8.0\bin
  2. win+r，cmd启动命令提示符，mysql -u root -p 验证是否能在全局启动MySQL，能跑即环境配置完毕

MySQL命令行指令：

  1. mysql –version 查询版本号
  2. mysql -u root -p 登录mysql
  3. net stop/start mysql 关闭/开启mysql服务
